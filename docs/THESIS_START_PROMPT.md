## 🎯 AI活用の目的（重要）

### 1. 文書管理の完全性
- Gitによるバージョン管理（ミス防止）
- 変更履歴の完全記録
- 過去バージョンへの復元可能性

### 2. 多様なAI視点の導入
- 人間の視点・視座
- AI視点の導入による多角的分析
- 文系・社会学テーマへのAI知見活用

### 3. 複数AIでの協調執筆
- **Cursor**: 主要執筆、構成整理
- **ChatGPT**: アイデア展開、議論
- **Claude**: 深い分析、推論
- **Gemini**: 多角的視点
- **ClaudeCode**: 構造整理
- **複数AIが同一情報源（GitHub）にアクセス・編集・討論**

### 4. リサーチ管理
- 文献調査結果の記録
- 引用管理
- データソースの追跡

### 5. ハルシネーション回避
- 複数AIでの相互検証
- 事実確認プロセス
- 引用元の明確化

### 6. AI駆動での積極的執筆
- **方針**: ガンガン編集、ガッツリ進行
- **監督**: ユーザーが方向性決定
- **実行**: AIが執筆・編集

### 7. その他
- ファイル管理の自動化
- 生産性向上
- GitHub機能のフル活用（ブランチ、マージ、PR、MCP、gh CLI）
- PDCAサイクルでの改善知識蓄積
- エラー解決の委任
- 学習機会としての論文製作
