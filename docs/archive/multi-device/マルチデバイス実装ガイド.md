# マルチデバイス・マルチアカウント実装ガイド

## 🚀 概要

本ガイドは、マルチデバイス・マルチアカウントでのAI駆動開発統合システムの実装手順を詳細に説明します。

## 📋 前提条件

### 必要な環境
- **GitHubアカウント**: メインアカウント（0nyx-lab）、サブアカウント
- **Cursor**: 各デバイスにインストール済み
- **Python**: 3.8以上
- **Git**: バージョン管理システム

### 必要な権限
- **メインアカウント**: リポジトリのオーナー権限
- **サブアカウント**: リポジトリへの書き込み権限


## 🔧 Phase 3: マルチデバイス同期システムの実装

### 3.1 同期システムの実装

#### multi_device_sync.pyの配置
```bash
# ソースファイルの配置
cp src/multi_device_sync.py ./
```

#### 設定ファイルの作成
```bash
# 設定ファイルの作成
python3 -c "
import json
config = {
    'github_repo': '0nyx-lab/aidd-integration-system',
    'sync_interval': 300,
    'max_retries': 3,
    'devices': {
        'main-pc': {
            'device_name': 'Main PC (Windows)',
            'platform': 'Windows',
            'account': '0nyx-lab',
            'role': 'main'
        },
        'sub-pc': {
            'device_name': 'Sub PC (Mac)',
            'platform': 'macOS',
            'account': 'sub-account',
            'role': 'sub'
        }
    }
}
with open('multi_device_config.json', 'w') as f:
    json.dump(config, f, indent=2)
"
```

### 4.2 GitHub同期の設定

#### 自動同期スクリプトの作成
```bash
# sync_script.shの作成
cat > sync_script.sh << 'EOF'
#!/bin/bash
# マルチデバイス同期スクリプト

echo "🔄 マルチデバイス同期開始..."
cd /path/to/aidd-integration-system

# 仮想環境の有効化
source .venv/bin/activate

# 同期システムの実行
python3 multi_device_sync.py

# 変更のコミット・プッシュ
git add .
git commit -m "Multi-device sync: $(date)"
git push origin main

echo "✅ マルチデバイス同期完了"
EOF

# 実行権限の付与
chmod +x sync_script.sh
```

#### 定期実行の設定（cron）
```bash
# crontabの編集
crontab -e

# 5分ごとの同期（例）
*/5 * * * * /path/to/aidd-integration-system/sync_script.sh
```
